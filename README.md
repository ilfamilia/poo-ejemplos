# Ejemplos de Programación Orientada a Objetos en Java

Este repositorio contiene ejemplos prácticos de los conceptos fundamentales de la Programación Orientada a Objetos (POO) en Java.

Los ejemplos están organizados por temas para facilitar su comprensión y estudio.

## Temas incluidos

- Clases y objetos
- Encapsulamiento
- Herencia
- Polimorfismo
    - Sobrecarga
    - Inclusión
    - Genéricos

## Estructura

Los ejemplos están organizados en carpetas según el concepto que demuestran:

01-clases-y-objetos  
02-encapsulamiento  
03-herencia  
04-polimorfismo

Cada subcarpeta contiene ejemplos independientes que se pueden ejecutar individualmente.

## Cómo ejecutar los ejemplos

Cada ejemplo contiene una clase con método `main`.

Puedes ejecutarlos desde tu IDE o desde la terminal.

## Requisitos

- Java JDK 8 o superior

## Objetivo

Este repositorio sirve como material de apoyo para el aprendizaje de la Programación Orientada a Objetos en Java.
